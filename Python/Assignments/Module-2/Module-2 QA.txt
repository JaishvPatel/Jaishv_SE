Q14. What are negative indexes and why are they used?
-> Negative indexing retrieves elements from the end by providing negative numbers as sequence indexes. It starts from index -1 for the last list item, subtracting down in reverse fashion toward the first item. Negative indexing is used in Python to manipulate sequence objects such as lists, arrays, strings, etc.
